# Cyber-Security-Task5
**Task 5 Network Packet Analyzer **
The Network Packet Analyzer is a Python-based cybersecurity tool designed to capture and analyze live network packets in real time.
It displays essential information such as source and destination IP addresses, protocol types (TCP, UDP, ICMP), and optional payload data.
